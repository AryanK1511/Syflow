# Syflow Backend API

This is the main backend API for Syflow.

## Development Setup

### Prerequisites

- Python3
- Poetry
- pip install fastapi[standard]

### Commands

Clone the repo git clone git@github.com:AryanK1511/Syflow.git
cd Syflow/api
poetry install
poetry shell
fastapi dev api/main.py
fastapi run api/main.py
